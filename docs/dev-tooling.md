# Corker 개발 도구 설정

이 문서는 Corker 저장소에서 사용하는 기본 개발/리뷰 자동화 설정을 정리합니다.

## 포함된 설정 파일

- `.coderabbit.yaml`
  - CodeRabbit 리뷰 자동화 설정.
  - Auto-Claude 저장소의 기본 리뷰 정책을 동일하게 유지합니다.

- `.pre-commit-config.yaml`
  - Pre-commit 훅 설정.
  - Auto-Claude 저장소의 기본 훅 구성을 동일하게 유지합니다.

## 참고

- 본 설정은 저장소 정책과 일관성을 유지하기 위해 추가되었습니다.
- 필요 시 팀 정책에 맞게 훅/리뷰 프로필을 조정하세요.
